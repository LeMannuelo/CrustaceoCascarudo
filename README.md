# Crustáceo Cascarudo - Frontend

Este proyecto corresponde al **Frontend del Crustáceo Cascarudo**, desarrollado en **React + Vite**.  
Simula una tienda en línea de pedidos de comida, con funcionalidades de carrito de compras, gestión de pedidos y simulación de pago.

## Ejecución del proyecto

1. Instalar dependencias  
   ```bash
   npm install
2. Ejecutar el servidor de desarrollo
   ```bash
   npm run dev
